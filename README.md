# RAGpack

## A document chunker + preprocessor

### USE

- Go to [ragpack.top](https://www.ragpack.top).
- Choose your document chunking method + your intended retrieval method.
- Drag / select your documents and documentation.
- Test it out with the built in search functionality.
- Install your preferred zip file at the bottom of the page.
- Take the quickstart code.
- Make your LLM better know what you're trying to work with!
